矩阵的分块初等变换

下列三种变换称为分块矩阵的初等行变换:

​ (1)对换分块矩阵的两行;

​ (2)以可逆矩阵C左乘分块矩阵的某一行;

注:C的阶数与该行子矩阵的行数相等

​ (3)以矩阵C左乘分块矩阵的第i行加于第j行.

注:C的列(行)数与第i行(第j行)子矩阵的行数相等。

以上定义中的行换成列,左乘换成右乘,即得分块矩阵的初等列变换的定义.分块矩阵的初等行列变换也称为矩阵的分块初等变换.

经典求逆方法

分块求法

$$
\left[
\matrix{
A&0&E&0\
C&B&0&E
}
\right]-A^{-1}R1–B^{-1}R2–R2-B^{-1}CR1–>
\left[
\matrix{
E&0&A^{-1}&0\
0&E&-B^{-1}CA^{-1}&B^{-1}
}
\right]
$$

例5：

$$
求\left[
\matrix{
A&B\
0&D
}
\right]的逆矩阵
$$

$$
\left[
\matrix{
A&B&E&0\
0&D&0&E
}
\right]—>
\left[
\matrix{
E&0&A^{-1}&-A^{-1}BD^{-1}\
0&E&0&D^{-1}
}
\right]
$$

舒尔公式

设A为可逆矩阵，D为方阵，则有
$$
\left|
\begin{matrix}
A&B\
C&D\
\end{matrix}
\right|=
\left|
\matrix{
A
}
\right|*
\left|
\begin{matrix}
D-CA^{-1}B
\end{matrix}
\right|
$$
因为
$$
\left[
\matrix{
A&B\
C&D\
}
\right]–>
\left[
\matrix{
A&0\
0&D-CA^{-1}B\
}
\right]
$$
所以
$$
\left|
\matrix{
A&B\
C&D\
}
\right|=
\left|
\matrix{
A&0\
0&D-CA^{-1}B\
}
\right|=
\left|
\begin{matrix}
D-CA^{-1}B
\end{matrix}
\right|
$$
